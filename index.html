The Box Model

Browsers see every element contained in a rectangular box.

The Element Box

Every element in a document, both block-level and inline, generates a rectangular element box. The core, content area, of the element box is the content itself. The padding is the area held between the content area and an optional border. The border is a line that surrounds the element and its padding. The margin is an optional amount of space added on the outside of the border.

Specifying Box Dimensions

The default method applies the width and height to the content box. The resulting size of the element will be the dimensions specified plus the amount of padding and borders. The border-box method applies the width and height values to the border box, which includes the content, padding, and border. The resulting visible element box, including padding and borders, will be exactly the dimensions you specify.

Never apply a height

Images Move to the Background

Images that are used purely for decoration should be controlled with a style sheet rather than the markup. The benefit to specifying decorative images only in an external style sheet make the document cleaner and more accessible.

Alternative Text

Every img element must also contain an alt attribute that is used to provide a brief description of the image for those who are not able to see it. Alternate text (alt text) should serve as a substitute for the image content—serving the same purpose and presenting the same information.

If an image does not add anything meaningful to the text content of the page, leave the value of the alt attribute empty, null. There should be no character space between the quotation marks.

Other Visuals

iframe element

The inline frame lets you embed a separate HTML document or resource in a document. An iframe allows the owner of the content to control visual aspects of the content you are putting on your page.

video & audio elements

Different browsers support different video and audio formats. Ensuring your media is viewable across every combination of browsers, platforms, and devices you wish to reach, can be a complicated task.

canvas element

The canvas element creates an area on the web page for drawing with a set of JavaScript functions for creating lines, shapes, fills, text, animations, and so on. The illustrations are generated with scripting, it's dynamic, can draw things on the fly, and can respond to user input.
